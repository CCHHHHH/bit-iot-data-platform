## 🚀 使用方法

### 1. 主题切换

用户点击顶部导航栏的主题切换按钮，选择：
- ☀️ **浅色模式** - 明亮主题
- 🌙 **深色模式** - 深色主题
- 💻 **跟随系统** - 自动匹配系统主题

### 2. 使用新样式

在 Vue 组件中：

```vue
<template>
  <div class="iot-card">
    <div class="iot-card__header">
      <h3 class="iot-card__title">卡片标题</h3>
    </div>
    <div class="iot-card__body">
      <p>卡片内容</p>
    </div>
    <div class="iot-card__footer">
      <button class="iot-btn iot-btn--primary">主要按钮</button>
    </div>
  </div>
</template>

<style scoped>
/* 使用 CSS 变量 */
.my-custom-class {
  background-color: var(--color-bg-primary);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
}
</style>
```

### 3. 响应式设计

```css
@media (min-width: 768px) {
  .my-grid {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: var(--spacing-6);
  }
}
```

## 💡 最佳实践

### ✅ 推荐做法

1. **始终使用 CSS 变量**
   ```css
   /* ✅ 好 */
   color: var(--color-text-primary);
   
   /* ❌ 不好 */
   color: #212529;
   ```

2. **使用语义化类名**
   ```css
   /* ✅ 好 */
   .iot-btn--primary
   
   /* ❌ 不好 */
   .blue-button
   ```

3. **使用间距系统**
   ```css
   /* ✅ 好 */
   padding: var(--spacing-4);
   
   /* ❌ 不好 */
   padding: 16px;
   ```

4. **考虑深色模式**
   - 避免使用硬编码颜色
   - 测试两种主题下的效果
   - 确保足够的对比度

## 📝 维护说明

### 添加新颜色

在 `variables.css` 中添加：

```css
:root {
  --color-custom: #XXXXXX;
}

[data-theme='dark'] {
  --color-custom: #XXXXXX;
}
```

### 添加新组件

1. 在 `src/styles/components/` 创建新文件
2. 在 `main.css` 中导入
3. 在规范文档中更新说明

### 修改现有样式

1. 确保两种主题都更新
2. 测试响应式布局
3. 验证可访问性
4. 更新文档
