## 四、技术规范

### 4.1 接口调用规范

```typescript
// 示例：获取插件列表
export function getPluginList(params?: { 
  current?: number
  size?: number
  name?: string
  status?: string
}) {
  return request.get('/api/integration/iot/plugin/list', { params })
}
```

### 4.2 错误处理规范

```typescript
try {
  const response = await api.getPluginList(params)
  if (response.data && response.data.code === 200) {
    // 处理成功响应
    plugins.value = response.data.data.records || []
    pagination.value.total = response.data.data.total || 0
  } else {
    ElMessage.error(response.data.message || '获取数据失败')
  }
} catch (error) {
  console.error('获取插件列表失败:', error)
  ElMessage.error('获取插件列表失败')
}
```

### 4.3 数据格式规范

**请求格式**:
```json
{
  "name": "插件名称",
  "version": "1.0.0",
  "description": "插件描述",
  "file": "文件对象"
}
```

**响应格式**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": "2026-03-25 10:00:00"
}
```
